# CheckTop Laptop Diagnostic Agent

## How to Use

1. Download the CheckTop Agent
2. Double-click `run-diagnostics.bat`
3. Wait for the diagnostics to complete (approximately 2-5 minutes)
4. Your report will be saved to:
   ```
   %APPDATA%\CheckTop Local Agent\last_report_human.json
   ```

## Requirements

- Windows 10 or Windows 11
- No installation needed!
- Runs completely offline

## What It Checks

- Battery Health
- CPU Performance & Temperature
- Storage Health (SMART data)
- Display for dead pixels
- USB, HDMI, and Audio ports

## Troubleshooting

- If you get a security warning, click **"More info"** then **"Run anyway"**
- SMART disk checks require Administrator privileges
- All tests are safe and non-destructive
